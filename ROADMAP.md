# NotifyHub Roadmap

The roadmap is directional and may change as platform APIs, contributor capacity, and user feedback evolve.

## Foundation

- [x] Modular TypeScript monorepo
- [x] English and French i18n foundation
- [x] Discord Components V2 helpers
- [x] Railway health checks and graceful shutdown
- [x] Open-source contribution and security files

## First usable release

- [ ] Persistent guild and subscription configuration
- [x] Twitch EventSub ingestion and normalization
- [ ] Notification deduplication
- [ ] Delivery history and retry policy
- [ ] Setup and test slash commands

## Platform expansion

- [ ] YouTube uploads and live streams
- [ ] Kick live notifications
- [ ] Bluesky posts
- [ ] TikTok posts and live events, subject to API availability
- [ ] X posts, subject to API access and operating cost

## Operations

- [ ] PostgreSQL storage
- [ ] Redis-backed job queue
- [ ] Metrics and structured audit events
- [ ] Dashboard and OAuth connection management
